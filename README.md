PLANO DE TESTES TAREFA CALCULADORA


1-Objetivo dos testes :
Será validado a funcionalidade de uma calculadora, sendo que suas funções são Somar,Dividir,Subtrair e Multiplicar, sendo que seu teste é necessário para saber sem as tarefas estão performando conforme pedido.

2-Será utilizada boas práticas referente ao projeto como, Testes manuais, testes funcionais e validação de limites.

3- Incluídos serão soma, divisão, subtração e multiplicação já os excluídos serão números 0,5,6,7,8,9 dígitos, termos científicos e histórico de consulta.

4-Tipos de testes são zerar os números, fazer contas de forma práticas, regressão de números.

5-Testes manuais no navegador, testes repetidos após alterção do código.

6- Navegador : chorme/edge  linguagem :javascript ambiente: web editor: vscode

7- Dev: Implementar dados, testar funcionalidades e corrigir possíveis erros

8- Risco de digitar números como 0,5,6,7,87,9 possíveis erros em questões grandes e suas contigências seriam validar entrada antes do calculo, mostrar mensagem de erro e bloquear entradas acima de 4 dígitos.

9- Navegador chorme, GitHub, vscode

10- testes executados com sucesso e correções realizadas
